Detecting Humans in RGB-D Data with CNNs
K. Zhou, A. Paiement, M. Mirmehdi
IAPR Machine Vision Applications (MVA), 2017.
PDF / bibtex / ROI selector code / evaluation code

We present an RGB-D human detector that mainly consists of two components: (1) a depth-based ROI selector that generates detection proposals and (2) a two-stream CNNs for proposal classification in RGB-D data.
